## About the Music Library

This project makes information about the scores held in the [Lionel Hampton School of Music's](https://www.uidaho.edu/letters-arts-social-sciences/academics/music) Music Library available to explore and browse.

## Score Metadata

Each score in the library is cataloged by the student librarians using these fields:

- Title
- Composer
- Arranger
- Publisher
- Envelope
- Score Type
- Missing Parts?
- PlayingTime
- Comments 
- Library

The full information for each score is displayed on their Item page.
